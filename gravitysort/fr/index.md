## 1. Données collectées
- **Données de jeu (locales)** : progression, étoiles, pièces, série, succès, paramètres, skins, achats In-App actifs. Stockées sur ton appareil et en option dans ton iCloud privé.
- **Données publicitaires (Google AdMob)** : si tu donnes ton consentement via App Tracking Transparency (ATT) d'Apple, nous pouvons partager ton IDFA avec Google AdMob pour des pubs personnalisées. Sans consentement, les pubs sont contextuelles.
- **Achats In-App (StoreKit)** : traités par Apple. Nous recevons uniquement une confirmation de transaction.
- **Diagnostic** : MetricKit natif d'Apple. Local seulement, jamais envoyé à des tiers.

## 2. Données que nous ne collectons PAS
Email, nom, téléphone, géolocalisation, contacts, photos, caméra, micro, données personnelles identifiables.

## 3. Suppression des données
1. Supprime l'app (efface UserDefaults locaux)
2. Réglages iOS → iCloud → Gérer → Gravity Sort → Supprimer
3. Les achats In-App restent dans ton Apple ID, récupérables avec "Restaurer les achats"

## 4. Services tiers
Google AdMob, Google UMP, Apple iCloud, Apple StoreKit.

## 5. Enfants
Adapté tous publics (4+). Conforme COPPA, RGPD-K. AdMob filtre les pubs pour mineurs.

## 6. Tes droits
- **UE/UK/Suisse (RGPD)** : accès, rectification, effacement, portabilité, opposition.
- **Californie (CCPA)** : nous NE vendons PAS tes données.

## 7. Sécurité
Sandbox iOS, HTTPS, chiffrement iCloud Apple.

## 8. Conservation
Tes données restent jusqu'à ce que tu les supprimes (section 3).

## 9. Modifications
Nous mettons à jour la date. Les changements matériels sont notifiés dans l'app.
