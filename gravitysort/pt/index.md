## 1. Dados que recolhemos
- **Dados de jogo (locais)**: progresso, estrelas, moedas, sequência, conquistas, configurações, skins, IAPs ativos. Armazenados no teu dispositivo e opcionalmente no teu iCloud privado.
- **Dados de publicidade (Google AdMob)**: se deres consentimento via App Tracking Transparency (ATT) da Apple, podemos partilhar o teu IDFA com o Google AdMob para anúncios personalizados. Sem consentimento, anúncios são contextuais.
- **Compras In-App (StoreKit)**: processadas pela Apple. Recebemos apenas confirmação da transação.
- **Diagnóstico**: usamos MetricKit nativo da Apple. Local apenas, não enviado a terceiros.

## 2. Dados que NÃO recolhemos
Email, nome, telefone, localização, contactos, fotos, câmara, microfone, dados pessoais identificáveis.

## 3. Eliminação de dados
1. Apaga a app (limpa UserDefaults locais)
2. iOS → iCloud → Gerir → Gravity Sort → Eliminar dados
3. IAPs ficam no teu Apple ID, recuperáveis com "Restaurar compras"

## 4. Serviços de terceiros
Google AdMob, Google UMP, Apple iCloud, Apple StoreKit.

## 5. Crianças
Apto para todas as idades (4+). Cumprimos COPPA, GDPR-K. AdMob filtra anúncios para menores.

## 6. Os teus direitos
- **EU/UK/Suíça (RGPD)**: acesso, retificação, supressão, portabilidade, oposição.
- **California (CCPA)**: NÃO vendemos os teus dados.

## 7. Segurança
Sandbox iOS, HTTPS, encriptação iCloud da Apple.

## 8. Retenção
Os teus dados ficam até os apagares (secção 3).

## 9. Alterações
Atualizamos a data. Mudanças materiais são notificadas in-app.
